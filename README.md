# DigitClassifier

A machine learning project that classifies handwritten digits using logistic regression. The project includes accuracy evaluation and confusion matrix visualization to analyze model performance.

## Dataset

The dataset consists of 8x8 images of handwritten digits (0-9) from the sklearn.datasets module.

## Features

- Logistic Regression model for classification
- Split dataset into training and test sets
- Accuracy calculation
- Confusion Matrix visualization
- Sample digit image display
